\# 🧠 AI Life OS



\### 🚀 AI-Powered Productivity Dashboard



AI Life OS is an \*\*all-in-one productivity web application\*\* that helps users manage their \*\*tasks, projects, deadlines, and performance\*\* — enhanced with an integrated \*\*AI Assistant\*\*.



👉 It combines:



\* 📊 Data visualization

\* 📅 Calendar scheduling

\* 🏆 Gamified productivity (points system)

\* 🤖 AI assistance



—all in one unified dashboard.



\---

✔ Combines \*\*Productivity + Analytics + AI\*\*

✔ Clean, modern dashboard UI

✔ Persistent user data (login system)

✔ ChatGPT-style AI assistant with memory

\## 🧩 Features



\### 📊 Dashboard



\* Overview of all tasks:



&#x20; \* Planner tasks

&#x20; \* Deadline tasks

&#x20; \* Project tasks

\* Displays:



&#x20; \* Total tasks

&#x20; \* Completed tasks

&#x20; \* Pending tasks

&#x20; \* Progress percentage

\* Includes:



&#x20; \* Pie chart (task distribution)

&#x20; \* Bar graph (comparison)

&#x20; \* Calendar view



\---



\### 📝 Planner



\* Add daily tasks

\* Mark tasks as completed or pending

\* Delete tasks

\* Real-time updates on dashboard



\---



\### 📂 Projects



\* Add projects with deadlines

\* Track completion status

\* Earn \*\*+15 points\*\* per completed project

\* Auto-added to calendar



\---



\### ⏰ Deadlines



\* Assign tasks to specific dates

\* Calendar-based planning

\* Mark deadlines as completed

\* Earn \*\*+10 points\*\* per completed deadline



\---



\### 📈 Performance Tracking



\* Displays productivity score (%)

\* Graph showing progress trend

\* Minimal and clean UI



\---



\### 🏆 Points System



\* Gamified productivity tracking

\* Encourages consistent task completion



\---



\### 🤖 AI Assistant



\* Ask coding, study, or general questions

\* ChatGPT-style interface (chat bubbles)

\* Stores chat history per user

\* “Recent Questions” panel

\* Powered using Groq (LLaMA models — free AI)
